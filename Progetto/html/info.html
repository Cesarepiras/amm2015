Procedura di consegna

  • Il progetto si consegna tramite il server pubblico, che avete configurato a lezione. Se non lo avete ancora fatto, cercate di verificare che il codice che inviate tramite il comando git sia effettivamente replicato sul server
  • Per replicare il database sul server pubblico dovete utilizzare la funzione esporta di phpMyAdmin sul vostro server locale e la funzione importa di phpMyAdmin sul server pubblico. Nella sezione MySQL del corso trovate istruzioni più dettagliate
  • L'unica versione che proverò del vostro progetto è quella sul server pubblico.
    • Corollario: Non accetto codice consegnato via email, o in qualsiasi altro modo che non sia quello indicato.
